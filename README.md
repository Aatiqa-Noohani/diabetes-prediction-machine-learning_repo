<h1 align="center">🩺 Diabetes Prediction using Machine Learning</h1>

<p align="center">
  A Machine Learning project that predicts whether a person is diabetic or not using the PIMA Indians Diabetes Dataset and Support Vector Machine (SVM).
</p>

<hr>

<h2>📌 Project Overview</h2>

<p>
This project uses the <b>PIMA Diabetes Dataset</b> to build a predictive model for diabetes diagnosis.
The dataset is preprocessed, standardized, and then used to train a Support Vector Machine (SVM) classifier.
</p>

<h2>🚀 Features</h2>

<ul>
  <li>Data Collection and Analysis</li>
  <li>Data Preprocessing</li>
  <li>Feature Standardization using StandardScaler</li>
  <li>Train-Test Split</li>
  <li>Support Vector Machine (SVM) Classification</li>
  <li>Model Evaluation using Accuracy Score</li>
  <li>Prediction System for New Patient Data</li>
</ul>

<h2>📊 Dataset Information</h2>

<p>The dataset contains the following medical attributes:</p>

<ul>
  <li>Pregnancies</li>
  <li>Glucose</li>
  <li>Blood Pressure</li>
  <li>Skin Thickness</li>
  <li>Insulin</li>
  <li>BMI</li>
  <li>Diabetes Pedigree Function</li>
  <li>Age</li>
  <li>Outcome (0 = Non-Diabetic, 1 = Diabetic)</li>
</ul>

<h2>🛠️ Technologies Used</h2>

<ul>
  <li>Python</li>
  <li>Pandas</li>
  <li>NumPy</li>
  <li>Scikit-Learn</li>
  <li>Jupyter Notebook</li>
</ul>

<h2>📂 Project Workflow</h2>

<ol>
  <li>Import Libraries</li>
  <li>Load Dataset</li>
  <li>Data Analysis</li>
  <li>Separate Features and Labels</li>
  <li>Standardize Data</li>
  <li>Split Data into Training and Testing Sets</li>
  <li>Train SVM Classifier</li>
  <li>Evaluate Model Accuracy</li>
  <li>Predict Diabetes for New Input Data</li>
</ol>

<h2>📈 Model Performance</h2>

<p>
The model is trained using a <b>Linear Support Vector Machine (SVM)</b> and evaluated using accuracy scores on both training and testing datasets.
</p>
